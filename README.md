# INSTALLATION
1.	git clone https://github.com/ZeldaAnnisa/shoshop-onlineshopping.git
2.	composer update
3.	composer install
4.	php artisan migrate
5.	php artisan db:seed
6.	php artisan key:generate
7.	php artisan serve
